    Install

  • Download and unzip the archive using WinRar or Zip
  • If you have Minecraft open, you need to close it
  • Click start, type %appdata%/in the search bar.minecraft/versions
  • If you do not have such a folder, it must be created
  • Transfer the file .jar with replacement close all Windows
  • Log in to Minecraft, in the settings, select a profile called ht+Rewrite
  • Save it and then log in under this profile
